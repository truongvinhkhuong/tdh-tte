# SEO Content Guidelines

Hướng dẫn viết content tối ưu SEO cho TTE Website.

---

## 📝 Nguyên tắc chung

### 1. Title (Tiêu đề)

- **Độ dài**: 50-60 ký tự (hiển thị đầy đủ trên Google)
- **Format**: `[Keyword chính] - [Mô tả ngắn]`
- **Ví dụ tốt**: `Van điều khiển Fisher - Giải pháp cho ngành dầu khí`
- **Ví dụ xấu**: `Fisher` (quá ngắn, không mô tả)

### 2. Meta Description

- **Độ dài**: 150-160 ký tự
- **Phải có**: Call-to-action, keyword chính
- **Ví dụ tốt**: 
  > Khám phá van điều khiển Fisher với công nghệ tiên tiến. Toàn Thắng cung cấp giải pháp hoàn chỉnh cho ngành dầu khí và hóa dầu.

### 3. Headings (H1, H2, H3)

```
H1: Tiêu đề chính (chỉ 1 per page)
  H2: Các section chính
    H3: Sub-sections
```

- H1 phải chứa keyword chính
- Không skip levels (H1 → H3)

## 🖼️ Hình ảnh

### Tên file

- ✅ `may-bom-cong-nghiep-flowserve.jpg`
- ❌ `IMG_20240115.jpg`

### Alt Text

- Mô tả ngắn gọn nội dung ảnh
- Chứa keyword nếu tự nhiên
- **Ví dụ**: `Máy bơm công nghiệp Flowserve dòng ERPN`

### Kích thước

| Loại | Kích thước | Định dạng |
|------|------------|-----------|
| Hero/Banner | 1920x1080 | WebP/AVIF |
| Product | 800x800 | WebP |
| OG Image | 1200x630 | JPG |

## 🏷️ Cho từng loại content

### Sản phẩm (Product)

| Field | Yêu cầu SEO |
|-------|-------------|
| Name | Chứa brand + loại sản phẩm |
| Short Description | 150-160 ký tự, có keyword |
| Description | Đầy đủ, structured với headings |
| Images | Alt text đầy đủ |

**Template tên sản phẩm:**
```
[Loại sản phẩm] [Thương hiệu] [Dòng/Model]
VD: Van điều khiển Fisher DVC6200
```

### Dự án (Project)

| Field | Yêu cầu SEO |
|-------|-------------|
| Title | Tên dự án + vị trí |
| Challenge | Mô tả vấn đề cần giải quyết |
| Solution | Nhấn mạnh giải pháp TTE |
| Products | Link đến products liên quan |

### Dịch vụ (Service)

| Field | Yêu cầu SEO |
|-------|-------------|
| Title | Keyword dịch vụ + giá trị |
| Description | Benefits đầu tiên, chi tiết sau |
| Icon | Chọn icon phù hợp ngành |

## 🌐 Đa ngôn ngữ

### Nguyên tắc

1. **KHÔNG dịch máy** - Phải có người review
2. Giữ **ý nghĩa** chứ không dịch từng từ
3. Tối ưu **keyword từng ngôn ngữ**

### Ví dụ

| Tiếng Việt | Tiếng Anh |
|------------|-----------|
| Van công nghiệp | Industrial Valves |
| Máy bơm dầu khí | Oil & Gas Pumps |
| Dịch vụ bảo trì | Maintenance Services |

---

## ✅ Checklist trước khi publish

### Content

- [ ] Title đủ dài và có keyword
- [ ] Meta description hấp dẫn
- [ ] Headings structured đúng
- [ ] Nội dung unique, không copy

### Media

- [ ] Tất cả images có alt text
- [ ] Tên file thân thiện SEO
- [ ] Kích thước tối ưu

### Technical

- [ ] Internal links đến pages liên quan
- [ ] External links có `rel="noopener"`
- [ ] Preview metadata đúng

### Đa ngôn ngữ

- [ ] Có version tiếng Anh
- [ ] Content được review, không dịch máy
